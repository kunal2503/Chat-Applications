# 💬 Real-Time Chat Application

A full-stack real-time chat application built with **Next.js (TypeScript)** on the frontend and **Node.js + Express** on the backend.  
It supports user authentication, one-to-one chat, real-time messaging, and scalable architecture.

---

## 🚀 Features

### Frontend (Next.js + TypeScript)
- ⚡ Built with Next.js (App Router / Pages Router)
- 🧠 TypeScript for type safety
- 🔐 Authentication (JWT-based)
- 💬 Real-time chat UI
- 👤 User list & chat list
- 🟢 Online / Offline user status
- 📱 Fully responsive design
- 🍪 Secure cookie-based auth handling
- 🌙 Optional Dark Mode

### Backend (Node.js + Express)
- 🚀 RESTful API with Express.js
- 🔐 JWT Authentication & Authorization
- 🔒 Password hashing with bcrypt
- 💬 Real-time messaging using Socket.IO
- 🗄️ MongoDB with Mongoose
- 📁 Clean MVC / Controller-based structure
- 🌐 CORS & cookie handling
- 🧩 Scalable & production-ready architecture
